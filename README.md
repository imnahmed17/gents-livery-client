# Gents-Livery-Server

## Technology Stack:
- React
- Next JS
- Recharts
- React Rating

## Project Overview:
Welcome to the Gents Livery. This project aims to develop the platform with enhanced features and UI/UX improvements tailored specifically for men's clothing enthusiasts. Leveraging technologies such as Next.js, and its features like Server-Side Rendering (SSR), Static Site Generation (SSG), Incremental Static Regeneration (ISR), the platform will serve as a dynamic marketplace for clothing items. This project emphasizes user engagement, seamless browsing, and an immersive shopping experience.

## Installation:
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Run the server using `npm run dev`.

### Endpoints:
- /products
- /products/:productsId
- /flash-sale
- /category
- /dashboard
- /dashboard/all-products
